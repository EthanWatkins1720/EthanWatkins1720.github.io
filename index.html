Assignments

Assignment 1 - Basic HTML

Create a basic HTML page layout on a topic of your choice. This page is about my trip to Japan. Elements used are of the most basic variety such as the title tag, h1, p, a bulleted list, and a small table. This assignment laid the very bottom groundwork for what followed without proper use of flexboxes.

Assignment 2 - Basic CSS

Add more style to your site and create a cohesive theme that will be used across the site. This assignment page was based on pirates because I found the pirate ipsum generator amusing. Many of the page elements are the same as the first but with a little more consideration put into style, format, and technical application.

Assignment 3 - Page Layout

Recreate a template laid out using more advanced techniques. With a focus on the use of flexboxes, id's, classes, headers, and footers, the template is recreated here with minor style choices to keep it in line with the rest of the site. The use of flexboxes allows for more varied and unique layouts compared to a regular HTML layout.

Assignment 5 - Recreate CSS Page

Recreate a simplified version of a university website, excluding JavaScript elements. This page will mimic the layout of the university page using flexboxes to recreate the unique shapes and layout of the page.

Assignment 6 - Introduction to JavaScript

This assignment introduces JavaScript. We will learn about how JavaScript interacts with HTML and CSS as well as how it can be used to create unique, interesting, and interactive webpages.

Assignment 7 - If Statement Assignment

This assignment had us create a burger menu that appeared only on lower resolution screens. This menu would toggle the main nav menu. This, along with the exercises present on the page, introduced JS if statements.

Assignment 8 - Loops

This assignment had us use loops to create a running man who moves from the left to the right using an expanding margin. We also created an improved version of the thermometer from the previous assignment.

Assignment 9 - Arrays

This assignment had us use arrays to store a selection of quotes to be displayed advanced colors to make a rainbow. In the end, a pot of gold will appear over the rainbow as well. The timeout function was also used to make the rainbow appear slowly.

Assignment 10 - Classes

This assignment had us use an array to store classes to create and display a number of toys. Each toy has a selection of details that are displayed when the users hovers over the image of the toy.

Project Part 1

Part 1 of the semester project. A wireframe created using MockITT to lay out a news website/forum called A-Z Tech. The site is intended as a news site for professional news reporting and a forum for the tech community to discuss technology.

Project Part 2

Part 2 of the semester project. The previously made wireframe is recreated using HTML and CSS to create a layout to be built on later. Placeholder images and lorem ipsum is used in place of actual media and text for this part.

Project Part 3

Part 3 of the semester project. The previous layout is improved, building on the overall style of the site, replacing the lorem ipsum with actual text, and adding actual images where needed. Layouts are also optimized for small screens/mobile.
